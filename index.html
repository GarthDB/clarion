Simplifying Your Web Development Process

Simpler CSS

Using the Clarion Style framework you can easily create scalable and maintainable style sheets. This will reduce complexity and reduce development time.

Easy Scaffolding

Using the CLI, you can scaffold out projects and create new integrated style sheets with no additional work.

Use Your Favorite Tools

Clarion makes it easy for you to use your favorite tools like SASS, LESS, Gulp, and WebPack.

Please note that this project is still in alpha as features are still being added. But, do not let it deter you from taking advantage of its great capabilities.

Terminal: New Project Quick-Steps
>npm install -g clarion
>clarion new MyProject
>cd MyProject
>npm install
>npm run dev
Get started in just a few steps!

Your New Project

The default project is configured with SASS using the .scss syntax and webpack as the compiler and module manager.

The project architecture implements the Clarion Style Architecture.

Adding a New File

Additional style files can easily be manged through the CLI as well. In your terminal type the following command:

clarion add element headings

This will create the file _headings.scss in the 03_Elements directory as well as add @import '_headings.scss'; import statement to the directory manifest file so it can be included in your final CSS file.

Removing a File

Similar to adding a file, removing files can also be done through the CLI.

clarion remove element headings

This will remove the file _headings.scss in the 03_Elements directory as well as remove @import '_headings.scss'; import statement from the directory manifest file.

Building Your Project for Production

To build your application for final use, run the build command.

npm run build

The final compiled JavaScript and CSS file are in build directory in the root of your project.
