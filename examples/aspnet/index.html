Clarion and ASP.Net

Clarion fits in nicely with your ASP.Net projects. Managing Font, Style and Script files in your presentation layer can often become complex and cluttered very quickly. Clarion will help keep your project clean.

Add the Architecture

In the command line, navigate to to the project folder you would like to add Clarion to. (If you are using Visual Studio 2017, you right-click the project and select "Open in Command Line".)

Run the following command to install the basic architecture:

clarion new

Include New Files

The first thing that needs to be done is to include the new files and folders in the project.

  1. At the top of the Solution Explorer window select the "Show All Files" button.
  2. Right-click the following directories and files and select "Include In Project":
    • build
    • src
    • package.json
    • postcss.config.js
    • webpack.config.js
  3. You can delete the index.html file.

Update Directories

Now that the files have been added to the project, there are 2 things that need to be updated:

  1. The added directories probably do not match the project's naming conventions.
  2. Files added using the CLI will not be included in the project and will have to be included manually.
